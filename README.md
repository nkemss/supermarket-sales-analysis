# Supermarket Sales Analysis

## Overview
This project explores supermarket sales data to identify trends in customer purchasing behavior and payment preferences.

## Objectives
- Analyze sales performance across different customer transactions.
- Examine the distribution of payment methods used by customers.
- Identify patterns and insights that may support business decision-making.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Payment method analysis
- Sales trend visualization
- Insight generation

## Key Insights
- Identified the most frequently used payment methods.
- Explored relationships between payment methods and sales transactions.
- Highlighted customer purchasing patterns within the dataset.

## Files
- `supermarket_sales_analysis.ipynb` – Main analysis notebook
- `supermarket_sales.csv` – Dataset used for analysis
